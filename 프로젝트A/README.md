# 실시간 마케팅 반응고객 예측 및 감지조건 최적화
----

## 프로젝트 개요
```
· 트렌드가 빠르게 변화하는 유통업에서는 신규 고객을 확보하기 위해 계열사 이용 고객 대상으로 실시간 마케팅을 수행하고 있음.
· 실시간 마케팅을 효과적으로 활용하기 위해 데이터 분석 및 예측모델을 수행하려 함
(1) 구매가능성이 높은 고객군을 예측
(2) 최적의 감지조건의 규칙을 찾고 영업이익을 최대화하려 함 
```
